Поместите сюда фотографии работ для блока "Наши работы".

Используйте такие имена файлов:

- `work-01.jpg`
- `work-02.jpg`
- `work-03.jpg`
- `work-04.jpg`
- `work-05.jpg`
- `work-06.jpg`

Если фотографий больше или нужны другие подписи, обновите секцию `#portfolio` в [../../index.html](../../index.html).

Рекомендации:

- Вертикальный или близкий к вертикальному кадр смотрится лучше всего.
- Оптимальная ширина: от `1200px`.
- Формат: `jpg`, можно заменить пути в `index.html` на `webp`, если потребуется.
